# MediScan AI 🏥✨

![MediScan AI Cover](frontend/public/favicon.ico) <!-- Placeholder, replace with actual banner if available -->

MediScan AI is a comprehensive, production-ready digital health screening platform. It empowers users with AI-driven preliminary clinical assessments using computer vision, natural language processing, and risk-scoring algorithms. 

> **⚠️ IMPORTANT CLINICAL DISCLAIMER:** MediScan AI is generated by an AI-powered screening tool and is **not a substitute for a medical diagnosis**. It is intended solely as a preliminary screening aid to assist healthcare professionals. All findings must be reviewed and validated by a licensed physician before any clinical decisions are made.

## 🌟 Key Features

- **Multi-Modal AI Screening**:
  - **🩻 Chest X-Ray Analysis**: Detects pulmonary anomalies, congestion, and infiltrations using `torchxrayvision` (DenseNet) and Vision Transformers (ViT).
  - **🔍 Skin Disease Detection**: Analyzes dermatological photographs for potential lesions and atypical patches.
  - **🩸 Diabetes Risk Calculator**: Evaluates metabolic scores based on BMI, age, and clinical symptoms.
  - **🧠 Mental Health Text Analysis**: Scans free-form text for emotional patterns, distress, and crisis flags using NLP.
- **⚡ Persistent AI Microservice**: Python Flask backend pre-loads massive PyTorch/Transformer weights on startup, enabling lightning-fast (1-2s) inference times.
- **📊 Comprehensive Clinical Reports**: Generates highly detailed, print-ready triage reports with priority thresholds, risk bands, and demographic tracking.
- **🌓 Modern Premium UI**: Beautiful Dark/Light mode interface featuring glowing cyan accents, glassmorphism, and the geometric `Outfit` font.
- **🌍 Bilingual Support**: Seamlessly toggle between English (EN) and Urdu (UR).
- **🔒 Secure Architecture**: Robust JWT-based authentication with encrypted passwords and PostgreSQL data persistence.

## 🏗️ Architecture & Tech Stack

The project is structured as a robust three-tier architecture:

### 1. Frontend (Client)
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS v4, custom Dark/Light theming
- **Routing**: `wouter`
- **Icons & UI**: Lucide React, Radix UI primitives (Shadcn UI), Recharts for trend mapping.
- **State Management**: React Query (`@tanstack/react-query`), Context API.

### 2. Backend (Node.js API)
- **Framework**: Express.js (TypeScript)
- **Database**: PostgreSQL (via `pg` pool)
- **Security**: JWT (`jsonwebtoken`), `bcrypt` for password hashing, CORS.
- **Role**: Handles user authentication, database operations, and acts as a secure proxy to the Python AI microservice.

### 3. AI Microservice (Python)
- **Framework**: Flask
- **Machine Learning**: PyTorch, Hugging Face `transformers`, `torchxrayvision`.
- **Role**: A persistent backend that loads heavy ML models directly into memory on startup and exposes fast HTTP endpoints (`/analyze/xray`, `/analyze/skin`, etc.) for the Node backend.

## 🚀 Getting Started

Follow these instructions to run MediScan AI locally on your machine.

### Prerequisites
- **Node.js** (v18+ recommended)
- **Python** (3.11+ recommended)
- **PostgreSQL** (running locally on port 5432)
- **Git**

### 1. Database Setup
Ensure PostgreSQL is running. Create a database named `mediscan`.
```sql
CREATE DATABASE mediscan;
```
*(Note: The Node.js backend will automatically initialize the required tables on the first run).*

### 2. Environment Variables
Create a `.env` file inside the `backend` directory:
```env
# backend/.env
DB_HOST=localhost
DB_PORT=5432
DB_NAME=mediscan
DB_USER=postgres
DB_PASSWORD=your_database_password
JWT_SECRET=your_super_secret_jwt_key
PORT=5000
PYTHON_SERVICE_URL=http://localhost:5001
```

### 3. Install Dependencies
Open a terminal and install dependencies for both Node environments:
```bash
# Install Backend Dependencies
cd backend
npm install

# Install Frontend Dependencies
cd ../frontend
npm install
```

### 4. Setup Python Microservice
Navigate to the AI models directory and install the required Python packages:
```bash
cd mediscan-models
pip install -r requirements.txt
# (Ensure torch, transformers, flask, flask-cors, torchxrayvision, etc., are installed)
```

## 🏃‍♂️ Running the Application

To achieve the 1-2 second AI inference speeds, you **must start the Python microservice first** so it can pre-load the model weights.

### Terminal 1: Start AI Microservice
```bash
cd mediscan-models
python server.py
```
*Wait until the console indicates all models (X-Ray, Skin, Diabetes, Mental Health) have successfully loaded into memory (runs on port 5001).*

### Terminal 2: Start Node.js Backend
```bash
cd backend
npm run dev
```
*The backend will connect to PostgreSQL, initialize tables, and listen on port 5000.*

### Terminal 3: Start React Frontend
```bash
cd frontend
npm run dev
```
*The frontend will be available at `http://localhost:5173`. Open this URL in your browser.*
